Grunt plugin to output of the original & gzipped size of a string or buffer:

```sh
Running "filesizegzip" task
>> ./assets/dist/js/script.js 20.2kb → 6.8kb (gzip)

Running "filesizegzip" task
>> ./assets/dist/css/kickoff.css 57.06kb → 13.27kb (gzip)
```
